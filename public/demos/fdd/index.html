FDD Update Engine

Every franchisor must refile a Franchise Disclosure Document each year. Today a paralegal reads last year's filing line by line to find what changed. This engine does the first pass: it diffs last year's FDD against this year's, item by item, and surfaces every change an attorney must review — each one quoted verbatim from the source filing, never paraphrased, never invented.

How the verbatim guarantee works: the model only proposes changes; a deterministic filter then discards any quoted value that is not a literal substring of the real filing. It cannot invent a fee or a date. Items 9, 17, 20, 23 bypass the model entirely — they are computed (Item 20's outlet counts are pure arithmetic on the roster).

Items 3, 4, 17, 19, 21, 22, 23 always require licensed-attorney review per NASAA Universal Rule 7. This demo is for evaluation only — not a substitute for counsel.
