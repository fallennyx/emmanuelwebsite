This is the first signal from my corner of the multiverse. _Replace this with your
own writing._

I built this site to do a few things in the open: think out loud, build in public,
and keep everything I make — essays, research, projects — in one orbit.

## How entries work

Every entry is a Markdown file in `src/content/archive/`. Set its `category`
(`writing`, `project`, or `research`) and it shows up under the matching filter.

> No database, no admin panel. A file is an entry.
